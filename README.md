# Hope-Speech-Detection-in-Code-Mixed-Tulu-Language-in-DravidianLangTech
SSN_HopeNetters submission for the Shared Task on Hope Speech Detection in Code-Mixed Tulu Language (DravidianLangTech @ ACL 2026). This repository contains XLM-RoBERTa-based models for coarse-grained and fine-grained hope speech classification.
